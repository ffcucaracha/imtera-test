# Инструкция по использованию сервиса

Сервис предназначен для владельцев организаций, которым важно быстро просматривать отзывы клиентов и управлять ссылкой на свою организацию в Яндекс Картах.

## Возможности
- Регистрация и вход в личный кабинет  
- Добавление ссылки на организацию в Яндекс Картах  
- Автоматическое получение и отображение отзывов  
- Просмотр отзывов в удобном интерфейсе  

## Начало работы

### 1. Регистрация
1. Откройте главную страницу сервиса.  
2. Нажмите «Регистрация».  
3. Укажите адрес электронной почты и пароль.  
4. Войдите в личный кабинет после успешной регистрации.

### 2. Добавление организации
1. Перейдите в раздел «Настройка».  
2. В поле «URL организации» вставьте ссылку на вашу организацию в Яндекс Картах  
   (например: https://yandex.ru/maps/org/...).  
3. Сохраните изменения.

## Просмотр отзывов
1. Перейдите в раздел «Отзывы».  
2. После добавления ссылки сервис автоматически получает отзывы с Яндекс Карт.  
3. В разделе отображаются:
   - текст отзывов  
   - оценка  
   - автор  
   - дата отзыва  

Отзывы обновляются автоматически при повторном открытии раздела.

## Частые вопросы

### Нужно ли что-либо настраивать в Яндекс Картах?
Нет. Достаточно вставить ссылку на вашу организацию.

### Можно ли изменить ссылку позже?
Да. Вы можете в любой момент обновить URL в разделе «Настройки».
